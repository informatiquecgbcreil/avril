# Patch UX lot 1

Ce patch applique le premier lot de simplification UX :

- dashboard simple recentré sur les actions du quotidien ;
- bloc « Aujourd’hui et à reprendre » ajouté ;
- graphiques relégués plus bas en mode simple ;
- doublon de raccourci supprimé dans les actions rapides par défaut ;
- libellés de navigation plus humains ;
- repère rapide ajouté sur les pages principales ;
- page participants allégée avec recherche visible immédiatement ;
- filtres avancés repliés ;
- outils avancés participants déplacés hors de la vue standard ;
- bouton « Ouvrir » renommé en « Voir la fiche ».
